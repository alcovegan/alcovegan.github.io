Проекты

Некоторые коммерческие проекты, которые я сделал. В основном я специализируюсь на создании удобных сайтов по доставке еды. Также здесь мои личные проекты по авиабилетам.

Купить-Авиабилет.ru

kupit-aviabilet.ru

Сайт сделан для работы с партнерской программой Travelpayouts. Здесь было много работы с разными API и базой данных. Стояла задача сгенерировать максимальное количество страниц под низкочастотные запросы в тематике авиабилетов и собрать трафик. При этом страницы должны содержать какой-то контент, частью которого являются данные из API, часть - из баз данных с информацией о странах, городах, аэропортах, авиакомпаниях.

Что было сделано:
  • Разработка бэкенда (NodeJS, Express, MongoDB, Kue)
  • Разработка фронтенда (Bootstrap 4 + Nunjucks)
  • Деплой на сервер (sh-скрипты + nginx + pm2)

Что было сделано:
