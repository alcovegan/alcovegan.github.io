Кафе Dolce Vita

Что было сделано:

  • Разработка фронтенда и логики (VueJS)
  • Разработка конструктора пиццы (VueJS)
  • Деплой на сервер (nginx + pm2)

Доставка для сети кафе

delivery.california-cafe.ru

Доставка еды для сети армавирских кафе. Стояла задача разработать доставку еды с раздельными корзинами для каждого заведения после неудачной попытки заказчика реализовать то же самое с адекватной стоимостью на Iiko Widget.

Что было сделано:
  • Вёрстка (использовался фреймворк Bulma без особой кастомизации)
  • Разработка бэкенда и API (NodeJS, Express, Nodemailer)
  • Разработка фронтенда и логики (VueJS)
  • Реализация отправки уведомлений о заказе в Telegram и SMS
  • Деплой на сервер (nginx + pm2)
